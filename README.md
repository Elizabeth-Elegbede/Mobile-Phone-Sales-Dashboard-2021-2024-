# Mobile-Phone-Sales-Dashboard-2021-2024-
This Mobile Sales Dashboard is a comprehensive analytics tool, developed using Power BI, designed to track, visualize, and interpret mobile phone sales trends across India from 2021 to 2024. Built with a focus on clarity and actionable insights, the dashboard empowers stakeholders to make data-driven decisions in a rapidly evolving market.

---

## 📁 Project Overview
- How did revenue, cost, and profit evolve from 2021 to 2024?
- Which months generated the highest revenue in 2024?
- Which mobile brands contributed most to revenue and profit?
- Which cities delivered the highest profit margins?
- How do brand-level profit trends compare across the year?
- What strategic actions can be taken based on seasonal and regional sales patterns?

## 🧹 Data Preparation Steps
📂 Data Preparation Steps
✓ Cleaned missing values and inconsistent brand or city entries
✓ Standardized mobile brand names, city labels, and month formats
✓ Created a time dimension for monthly and yearly analysis
✓ Modeled relationships between sales, cost, profit, and brand/city dimensions
✓ Built DAX measures for:
- Total Revenue
- Total Cost
- Total Profit
- Profit Margin
- Year-over-Year growth metrics
- Brand-level and city-level aggregations
